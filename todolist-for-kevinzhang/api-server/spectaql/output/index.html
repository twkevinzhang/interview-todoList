When expected as an input type, any string (such as "4") or integer (such as 4) input value will be accepted as an ID.

Description

Field Name Description
id - ID!

owners - [User!]! 有多個執行人，也可能是空陣列
followers - [User!]! 有多個關注人，也可能是空陣列

due - RFC3339 到期日，可能因為沒有設定而是 null
description - String 內文
parentID - ID TodoItem 可以是一個巢狀的結構，parent 是父任務

children - [TodoItem!]! TodoItem 可以是一個巢狀的結構，children 是子任務
attachments - [Attachment!]! TodoItem 可以有多個附件，也可能是空陣列
comments - [Comment!]! TodoItem 可以有多個留言，也可能是空陣列
isCompleted - Boolean! 是否完成
createdAt - RFC3339! 建立時間
createdByUID - String! 建立者 UID
createdBy - User! 建立者
updatedAt - RFC3339! 最後更新時間
updatedByUID - String! 最後更新者 UID
updatedBy - User! 最後更新者
